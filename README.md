# PetShop OPS

宠物商店运营管理平台。

- 前端：Astro + neo-brutalist 风格
- 后端：Astro API Routes（TypeScript）
- 数据库：MySQL（mysql2 驱动）

## 页面

- `/` 仪表盘
- `/pets` 宠物档案
- `/inventory` 库存与仓位
- `/customers` 顾客与预约关系
- `/employees` 员工与负责关系
- `/preorders` 预约看板
- `/insights` 结构与接口说明

## API

- `/api/summary`
- `/api/pets`
- `/api/inventory`
- `/api/customers`
- `/api/employees`
- `/api/preorders`
